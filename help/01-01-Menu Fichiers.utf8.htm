Le menu Fichier

Ouvrir

Le menu <Fichier/Ouvrir> permet de choisir le répertoire qui contient le(s) jeux de données avec lequel travailler. S'il s'agit d'un répertoire qui contient des sous-répertoires de données, alors tous les sous-répertoires seront chargés. Sinon, seul le répertoire sélectionné sera chargé.

Ajouter

Le menu <Fichier/Ajouter> permet d'ajouter un jeu de données.

Ajouter un répertoire récent

Le menu <Fichier/Ajouter un répertoire récent> permet d'ajouter un jeu de données en proposant le choix parmi les dernier jeux de données chargés.

Fermer un jeu de données

Le menu <Fichier/Fermer un jeu de données> permet de fermer un des jeux de donnés chargés.

Fermer tout

Le menu <Fichier/Fermer tout> ferme tous les jeux de données.

Quitter

Le menu <Fichier/Quitter> ferme l'application
