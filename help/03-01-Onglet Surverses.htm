Onglet Surverses

La page permet de définir le scénario de surverse. Chaque point de surverse peut être activé/désactivé séparément, et sa plage de surverse spécifiée.

Une fois le scénario rempli, appuyez sur le bouton Affiche pour lancer le calcul. Les résultats sont alors disponibles dans les onglets Graphes" et Trajectoires".
